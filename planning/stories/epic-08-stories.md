# Epic 08 Stories: Tutor Portal

## Story 8.1: Tutor Authentication
**As a** Tutor
**I want to** log in securely
**So that** I can view my private pet data.

**Acceptance Criteria:**
- [ ] Separate Login Page for Tutors
- [ ] Auth provider restricted to "TUTOR" role
- [ ] Password Reset flow (Email link)

---

## Story 8.2: Tutor Dashboard & My Pets
**As a** Tutor
**I want to** see a list of my pets
**So that** I can select one to view details.

**Acceptance Criteria:**
- [ ] Fetch pets linked to logged-in user
- [ ] Display Pet Cards (Name, Photo, Next Appt)
- [ ] Responsive mobile-first layout

---

## Story 8.3: Digital Vaccination Card
**As a** Tutor
**I want to** view my pet's vaccine history
**So that** I have proof of vaccination on my phone.

**Acceptance Criteria:**
- [ ] Filter History events for "Vaccine"
- [ ] Show list: Date, Vaccine Name, Next Due Date
- [ ] highlight "Overdue" vaccines
